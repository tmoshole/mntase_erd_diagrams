How to read the diagrams

Each box represents a type of record, and the rows inside it describe the information kept for that record.

PK - Primary Key

The unique ID used to identify one record, like a member number.

FK - Foreign Key

A reference that connects a record to a related record in another box.

UK - Unique Key

A value that cannot be repeated, such as a public code or secure lookup value.

Connecting lines

These show how records relate, such as one member having many check-ins.

Circle and line symbols

A circle means optional, a line means one, and the fork shape means many.

Field types

uuid is an ID, string is text, int is a whole number, and date or timestamptz records time.
